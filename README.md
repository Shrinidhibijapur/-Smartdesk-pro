# SmartDesk Pro

**SmartDesk Pro** is a futuristic, glassmorphic web dashboard designed to boost your productivity with style. It combines essential tools like a Clock, Weather widget, Alarm system, Stopwatch, Timer, and To-Do list into a single, beautiful interface.

## Features

-   **Futuristic UI**: Glassmorphism design with Neon Glow effects.
-   **Dashboard**: Digital Clock, Date, Weather (Open-Meteo), and Quotes.
-   **Smart Alarms**: Set multiple alarms with custom labels and sound notifications.
-   **Circular Timer**: Visual countdown with multi-ring progress for Hours, Minutes, and Seconds.
-   **Stopwatch**: Precise timing with lap recording functionality.
-   **To-Do List**: Manage your daily tasks with a clean, animated list.
-   **Theme Support**: Switch between **Dark**, **Light**, and **Neon** modes.
-   **Persistence**: All your data (alarms, todos, settings) is saved automatically using LocalStorage.

## How to Use

1.  **Download** or **Clone** this repository.
2.  Locate the `index.html` file in the project folder.
3.  **Double-click** `index.html` to open it in your preferred web browser (Chrome, Edge, Firefox, etc.).
4.  Enjoy your new Smart Dashboard!

## Technologies Used

-   **HTML5**
-   **CSS3** (Variables, Flexbox, Grid, Animations)
-   **JavaScript** (ES6+, LocalStorage API)
-   **FontAwesome** (Icons)
-   **Google Fonts** (Outfit)
-   **Open-Meteo API** (Weather data)
